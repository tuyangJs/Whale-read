# 鲸阅言文 废弃的版本

这是一个搁置了半年的项目，现在发现代码有些屎山，所以不打算继续开发了。
准备重新开发这个程序，所以现在将它开源。
我并没有写编译脚本，如果你需要编译成exe，需要您自己实现。

## 我们更新了什么？

1. 我们优化并修复了，移动设备页面体验
2. 我增加了多国语言支持，自建了一套多语言系统，相比i18更加方便维护与便捷。
3. 我们优化了PC与移动端的导航栏体验。


## 安装依赖
```
$ cnpm i 
//or
$ npm i
```

## 启动项目

```
$ npm start
```
## 编译前端文件
```
$ npm run build 
```
